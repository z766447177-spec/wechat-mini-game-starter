# 微信小游戏独立游戏 Starter

这是一个从 0 到 1 的微信小游戏独立游戏初始化仓库。

## 当前目标
- 先搭一个最小可用的项目骨架
- 后续接入 Cocos Creator
- 之后再接入微信开发者工具和云开发
- 先做轻量 2D 原型，不一开始做大而全版本

## 当前技术路线
- 游戏引擎：Cocos Creator
- 微信侧调试：微信开发者工具
- 数据层：微信云开发（CloudBase）
- 素材工具：Pixelorama（Aseprite 平替）
- 文案/笔记：Markdown

## 下一步
1. 打开 Cocos Creator
2. 创建或导入实际项目
3. 配好微信小游戏构建
4. 建立存档、配置表、基础 UI
